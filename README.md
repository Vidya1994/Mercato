# Mercato

An e-commerce website built using Ruby on Rails.

## Versions

* ruby 2.4.0

* Rails 5.0.1

## Dependencies

* [Dropwizard backend service](https://github.com/susindaran/OnlineShopping) 

## Setup

* Download required gems
   ```
   bundle install
   ```

* Database initialization
   ```
   rake db:migrate`
   rake db:seed
   ```

* Run server
   ```
   rails server
   ```
